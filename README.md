# CSV Data Analyzer Web App 📊

Upload any CSV file and instantly get charts,
statistics, correlations and data quality reports.

## Live Demo
[Click here to try the app](https://csv-analyzer-webapp-x2gkyt3vtla4i74eahxwzq.streamlit.app)

## Features
- Dataset overview with column info and stats
- Interactive chart builder (5 chart types)
- Correlation heatmap and top pairs
- Missing values, duplicates and outlier detection

## Tools Used
- Python, Streamlit, Pandas, Matplotlib, Seaborn

## How to Run Locally
pip install streamlit pandas numpy matplotlib seaborn
streamlit run app.py
